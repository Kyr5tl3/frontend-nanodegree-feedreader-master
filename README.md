## Functionality

The following functionality has been tested:
  - RSS Feeds
      - are defined
      - have a "name" property
      - have a "url" property
  - The menu
      - is initially hidden
      - visibility is toggled on click
  - Initial entries
      - has at least 1 entry
  - New Feed Selection
      - has changed from the last feed
